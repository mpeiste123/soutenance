

# Soutenance AJC

## objectif
Les Chaudronneriees est un lieu de tournage/événementiel , nous exposons :

- Le problème : une planification manuelle + l'utilisation de Whatsapp --> erreurs
doublons, perte d'information et lenteur commerciale

- La Réponse : une plateforme ominicanale "low friction" qui conserve ls usagees en cours
(Google Calendar, WhatsApp) tout en apportant :

        -  un parcours Web qualifié (demande --> préréservation --> pipeline)
        -  un assistant vocal qui accélère la création et la modification 
           d'évènements et fiabilise le calendrier.
        -  Le pilotage: Objectifs SMART, KPIs, gouvernance/RACI,risques,budget,planning.
        -  outils numériques produits: tunnel testable (Web) vocalendar(vidéo + captures
           + commandes)


## Structure
- /docs/index.html : HUB (navigation)
- /docs/slides/ : PDF + viewer web
- /docs/tunnel/ : tunnel (pages/css/assets)
- /docs/vocalendar/ : démo + preuves

## Lancement rapide (local)
Ouvre docs/index.html dans un navigateur.
